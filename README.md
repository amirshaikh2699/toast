# Toast 
Toast made with Bootstrap and Vanilla JS 

## Installation
Copy style.css and Main.js into your project/Merge into existing main.js and Style.css

## Documentation
showToast(type,message,duration);

### Type
Datatype : string;
  1. "warning"
  2. "error"
  3. "success"
  
### Message 
Datatype : string;
  
### Duration
Datatype : number;
